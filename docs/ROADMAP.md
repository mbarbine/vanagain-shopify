# Roadmap

## Phase 1: Stabilize Launch Quality

- Keep the live storefront visually consistent across desktop, tablet, and mobile.
- Create the Shopify Admin policy/page records that theme fallbacks cannot create.
- Confirm Search & Discovery filters map to real product tags and collections.
- Add redirects for every retired legacy URL.
- Confirm Shopify Markets, shipping zones, duties, taxes, and currency behavior for the United States, U.S. territories, Canada, Mexico, Europe, and Australia.

## Phase 2: Improve Product Discovery

- Build a dedicated fitment finder using product tags and metafields.
- Add year/model/system filter chips above product results.
- Add collection landing copy for top systems: coolant, brakes, electrical, fuel, exhaust, camper, and transmission.
- Add product badges for OEM, aftermarket, rebuilt, kit, sale, limited stock, and core-charge items when the source data supports them.

## Phase 3: Increase Conversion

- Add fitment confidence panels on product pages.
- Add related kits and commonly bought together blocks.
- Add back-in-stock capture for out-of-stock products.
- Improve abandoned cart and browse abandonment email flows.
- Add review/testimonial snippets on high-traffic product and collection pages.

## Phase 4: Content and SEO Scale

- Reconcile all legacy blog posts against the SQL export and Shopify articles.
- Add internal links from technical articles to relevant parts collections.
- Add schema-rich FAQ sections for shipping, returns, coolant system, and fitment support.
- Produce guide pages for major repair categories and model year ranges.

## Phase 5: Measurement and Operations

- Add GA4 ecommerce events and Shopify analytics review dashboards.
- Track conversion rate, add-to-cart rate, search conversion, filter usage, AOV, repeat purchase rate, and email revenue share.
- Add weekly content/product QA checks for empty pages, broken links, missing product images, and missing descriptions.
